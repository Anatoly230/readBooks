### 9.1 Записи об окружающей среде

 `Environment Records`  — это тип спецификации, используемый для определения связи идентификаторов с конкретными переменными и функциями на основе лексической структуры вложенности кода ECMAScript. Обычно  `Environment Records`  связана с определённой синтаксической структурой кода ECMAScript, такой как FunctionDeclaration, BlockStatement или блок Catch в TryStatement. При каждом вычислении такого кода создаётся новая запись  `Environment Records`  для хранения привязок идентификаторов, созданных этим кодом.

Каждая  `Environment Records`   имеет поле `[[OuterEnv]]`, которое либо равно null, либо является ссылкой на внешнюю  `Environment Records`  . Это используется для моделирования логической вложенности значений записей среды. Внешняя ссылка на (внутреннюю)  `Environment Records`   — это ссылка на  `Environment Records`  , которая логически окружает внутреннюю  `Environment Records`  .   `Environment Records`  , конечно, может иметь свою собственную внешнюю  `Environment Records`  .  `Environment Records`   может служить внешней средой для нескольких внутренних записей среды. Например, если FunctionDeclaration содержит два вложенных FunctionDeclaration, то внешней записью среды для каждой из вложенных функций будет  `Environment Records`   для текущей оценки внешней функции.  `Environment Records`   — это чисто технические механизмы, которые не обязательно должны соответствовать какому-либо конкретному артефакту реализации ECMAScript. Программа на ECMAScript не может напрямую обращаться к таким значениям или управлять ими.

### 9.1.1 Иерархия типов записей об окружающей среде

 `Environment Records`  можно представить в виде простой объектно-ориентированной иерархии, где  `Environment Records`  — это абстрактный класс с тремя конкретными подклассами:
 1. декларативная  `Environment Records`
 2. `Environment Records`  объекта и глобальная  `Environment Records`
 3. `Environment Records`  функций
 4. `Environment Records`  модулей являются подклассами декларативной  `Environment Records` .


### 9.1.1.1 Записи декларативной среды
Каждая запись декларативной среды связана с областью видимости программы ECMAScript, содержащей объявления переменных, констант, let, классов, модулей, импорта и/или функций. Запись декларативной среды связывает набор идентификаторов, определённых объявлениями, содержащимися в её области видимости.
